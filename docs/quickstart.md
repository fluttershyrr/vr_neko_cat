# VR N.E.K.O.cat 快速开始

## 这是什么？

VR N.E.K.O.cat 是 N.E.K.O. 的 VR 插件。它通过 UDP 协议将位姿/手势/输入数据发送给 AnyaDance 驱动，再由 AnyaDance 注入 SteamVR，最终在 VRChat 中呈现全身追踪 (FBT)、手势、移动等行为。

```
N.E.K.O AI → 本插件 → UDP(127.0.0.1:39570) → AnyaDance → SteamVR → VRChat
```

---

## 前置条件

| 组件 | 说明 |
|------|------|
| Windows 10/11 | 操作系统 |
| N.E.K.O. | AI 伴侣平台，已安装并运行 |
| SteamVR | 已安装并完成房间设置 |
| VRChat | (可选) 通过 Steam 安装 |
| AnyaDance | 插件内置发布版，无需单独安装 |

---

## 第一步：启动插件

1. 打开 N.E.K.O. → 插件管理面板
2. 找到 **VR N.E.K.O.cat**，点击启用
3. 插件启动后自动扫描 AnyaDance 路径并准备就绪

## 第二步：启动驱动

在插件面板中点击 **一键启动**，或在 AI 对话框中直接说：

> "启动 VR 驱动"

插件会自动完成：启动 AnyaDance → 注册 SteamVR 驱动 → 启动 60Hz 数据流送。

## 第三步：进入 VRChat

1. 确保 SteamVR 状态灯为绿色（头显 + 基站就绪）
2. 启动 VRChat
3. 选择一个支持 FBT（全身追踪）的 Avatar
4. 你应该能看到虚拟身体跟随 AI 的位姿数据移动

## 控制面板

插件的 Web 控制面板提供完整的可视化操作：

| 区域 | 功能 |
|------|------|
| **设备位姿** | HMD / 左右手柄 / 髋部 / 左右脚 的 6 轴位姿调节 |
| **控制器输入** | 按键 / 扳机 / 握把 / 摇杆 / 五指弯曲 |
| **快捷手势** | 握拳 / 张开 / 指向 / 胜利 / 点赞 / Rock / 手枪 |
| **舞蹈** | VMD 文件加载与播放，支持变速和循环 |
| **动画** | 点头 / 摇头 / 歪头 / 挥手 / 鞠躬 / 待机呼吸 |
| **情感** | 12 种情感映射（开心/悲伤/愤怒/惊讶等） |
| **追踪** | HMD 注视目标或方向，平滑跟随 |
| **设置** | 网络 / 镜像 / 舞蹈 / 截图 参数 |
| **驱动** | 注册/注销/启动/停止/重启 SteamVR 驱动 |

右下角 ✨ 按钮可开启氛围粒子特效（飘雪 / 樱花 / 细雨 / 星尘），密度和速度自由调节。

## 常用配置

以下参数可在控制面板或配置文件中修改：

### 网络

| 参数 | 默认值 | 说明 |
|------|--------|------|
| UDP 主机 | `127.0.0.1` | AnyaDance 地址，本机无需修改 |
| UDP 端口 | `39570` | 与 AnyaDance 保持一致 |
| 流送率 | 60 Hz | 数据发送频率，降低可减少 CPU 占用 |

### 位姿与镜像

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 镜像双手 | 关闭 | 让左右手跟随同一侧数据对称运动 |
| 镜像双脚 | 关闭 | 同上，针对脚部追踪器 |

### VRChat

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 角色身高 | 1.5m | 用于 FBT 校准和位姿换算 |

## 故障排查

| 症状 | 可能原因 | 解决方法 |
|------|---------|---------|
| VRChat 中无追踪器 | 驱动未注册 | 点击"注册驱动"后重启 SteamVR |
| 位姿数据不动 | 流送未启动 | 点击"启动流送"或检查 UDP 端口冲突 |
| 控制器无反应 | 输入卡键 | 插件内置 5 秒自动释放，等待即可 |
| 舞蹈不播放 | VMD 路径无效 | 检查插件日志确认文件路径 |
| 截图返回空 | mss/PIL 未安装 | `pip install mss Pillow` |
| 驱动启动失败 | AnyaDance 路径错误 | 在设置中手动指定 AnyaDance 目录 |
| SteamVR 崩溃 | 驱动冲突 | 先卸载旧版驱动再重新注册 |

如以上方法无法解决，请查看 N.E.K.O. 日志输出，关键字过滤 `vr_neko_cat`。

## 高级功能

### 情感混合

```python
# AI 调用示例：70% 开心 + 30% 害羞
```

### VMD 舞蹈

支持标准 MMD 舞蹈文件 (.vmd)，插件自动将骨骼帧映射到 6 个 VR 追踪设备。加载后点击播放即可开始，支持循环和变速。

### AI 视觉桥接

启用自主视觉后，AI 可以：
- 检测 VRChat 中的玩家位置 → 自动注视
- 识别表情 → 映射为 VRChat 手势
- 观察游戏画面 → 做出相应反应
